# human-minigame
A project.

## Project uses:
* GLFW (multi-platform library for OpenGL): https://www.glfw.org/
* GLAD (GL loader): https://glad.dav1d.de/

## Get started:
```
git clone --recurse-submodules https://github.com/Murchik/human-minigame.git human-minigame
cd human-minigame
make configure
cd build
make
```
